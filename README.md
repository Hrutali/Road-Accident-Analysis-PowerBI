# Road Accident Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard for analyzing road accident data.
The dashboard helps identify accident trends, severity levels, and key risk factors.

## Key Insights
- Accident distribution by year and month
- Severity analysis (Fatal, Serious, Slight)
- Casualties by vehicle type
- Road surface and weather impact
- Urban vs Rural accident comparison

## Tools & Technologies
- Power BI
- Excel / CSV (Data Source)
- DAX for calculated measures

## Files in this Repository
- `Road_Accident_Analysis.pbix` – Power BI project file  
- `Road_Accident_Dashboard.pdf` – Exported dashboard for preview  

## How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Interact with filters and visuals
